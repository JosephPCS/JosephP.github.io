Warehouse Wreckage

A first-person game made with C++ and Unreal Engine blueprints where the player shoots different projectiles in a designated room. Made to simulate Rage Rooms. Players have limited amount of ammunition, but can shoot and destroy anything within the room. When the ammunition runs out the level restarts.

Digi-Dash

In a group project, made a video game recommendation website that randomly recommends games to users via a connection to the IGDB API. The API provides the information about the game, and we display it with a arcade-style aesthetic. The user can create an account and save recommended games to their profile. Worked primarily on the front-end using HTML,CSS and creating the UI.
